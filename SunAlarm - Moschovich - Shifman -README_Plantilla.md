## **`Objetivo`**

`El objetivo del proyecto es desarrollar un sistema de alarma inteligente para ventanas que pueda reaccionar de forma distinta según las condiciones ambientales y eventos detectados. Se busca brindar mayor seguridad aprovechando sensores combinados para monitorear luz ambiental, apertura de ventana y movimiento cercano.`

## **`Descripción del Proyecto`**

`El proyecto consiste en una alarma inteligente basada en Arduino que utiliza tres sensores: un LDR para detectar día o noche, un reed switch para identificar si una ventana está abierta o cerrada, y un sensor PIR para detectar movimiento. Dependiendo de la combinación de eventos, el sistema activa diferentes respuestas mediante LEDs y un buzzer, variando según sea de día o de noche. También incluye indicadores visuales fijos cuando no se detecta actividad.`

## **`Componentes`**

- `LDR (fotoresistencia)`  
- `Reed switch (sensor magnético de apertura)`  
- `Sensor PIR HC-SR501 (detector de movimiento)`
- `LED verde + resistencia 220 Ω`
- `LED rojo + resistencia 220 Ω`
- `Buzzer activo 5 V`
- `Arduino UNO o Nano`
- `Resistencia 10 kΩ (para la LDR)`
- `Protoboard, cables y fuente 5 V USB`

## **`Etapas del proyecto`**

- `Instalación y conexión de los sensores (LDR, reed switch, PIR)`
